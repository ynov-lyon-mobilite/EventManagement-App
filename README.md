# YVENT MOBILE APPLICATION


## Launch

From the right folder
```ionic serve``` with CMD
```npm start``` with VS Code terminal


## Deploying to iOS and Android

##### Capacitor Setup
```ionic build```
```ionic cap add android``` ou ```ionic cap add ios``` 
```ionic cap copy```
```ionic cap sync```

##### Android Deployment
```ionic cap open android```

##### iOS Deployment
```ionic cap open ios```
